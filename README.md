# JagmITCompany Dashboard

JAGM IT Company dashboard for operational tracking, data refreshes, and report publishing.

## What it does
This repository contains the dashboard assets and automation used to publish live status data, reports, and supporting pages for JAGM IT Company.

## Tech stack
- Python
- HTML
- JSON
- PowerShell
- GitHub Actions

## How to run
1. Clone the repository.
2. Run `python fetch-data.py` to refresh the dashboard data.
3. Open `index.html` or `landing.html` in a browser, or serve the folder with a static server.
